===========================================
שלב 1: הקמת שלד הפרויקט
זמן: 2025-01-19
===========================================

מה עשינו:
יצרנו פרויקט React חדש עם Vite ו-TypeScript. בחרנו ב-Vite
כי הוא הכי מהיר לפיתוח (Hot Module Replacement מיידי).
הוספנו Tailwind CSS v4 לעיצוב.

למה ככה:
- React כי זה הסטנדרט לאתרים מודרניים
- Vite במקום CRA כי הוא פי 10 יותר מהיר
- TypeScript כדי לתפוס שגיאות בזמן כתיבה
- Tailwind v4 לבנייה מהירה של UI בלי CSS נפרד

מה זה נותן לנו:
בסיס יציב עם DX מעולה - שינויים נראים מיד בדפדפן.

קבצים שנוצרו:
- package.json - הגדרות ותלויות
- vite.config.ts - הגדרות Vite
- tsconfig.json - הגדרות TypeScript
- tailwind.config.ts - הגדרות עיצוב
- index.html - נקודת הכניסה

מה הלאה:
נגדיר את מפרט העיצוב והפונטים.
