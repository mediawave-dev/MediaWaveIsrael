# Custom Fonts

Copy your Hebrew font files here:

## noa-shalev/
For headlines - elegant calligraphic serif
Expected files:
- NoaShalev-Regular.woff2
- NoaShalev-Regular.woff
- NoaShalev-Regular.ttf

## yarden/
For body text - friendly handwritten style
Expected files:
- Yarden-Regular.woff2
- Yarden-Regular.woff
- Yarden-Regular.ttf

---
Fallback font (Heebo) is installed via npm.
